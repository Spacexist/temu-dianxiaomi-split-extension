# 店小秘插件

Chrome MV3 扩展，用于辅助店小秘/商品列表页的批量操作和分类修正建议采集。

## 功能

### 1. 店小秘按数量选择

在店小秘「数据搬家 > Temu半托管数据」列表页，输入数量后自动勾选商品行。

- 输入 `100`，点击「按数量选择」
- 插件会从第一页开始选择前 100 个商品
- 当前页数量不够时，会自动点击「下一页」继续选择
- 日志会显示每页选择进度和失败原因

### 2. 商品分类修正建议采集

在商品分类错误列表页，点击「采集」后生成 Markdown。

采集字段：

- 商品标题
- SKC
- 原先错误类目
- 修改建议 1-3

插件会跳过带有「自动废弃 / 天后自动废弃」标记的商品，但日志中会保留调试信息。

## 安装方式

1. 打开 Chrome。
2. 进入 `chrome://extensions/`。
3. 打开右上角「开发者模式」。
4. 点击「加载已解压的扩展程序」。
5. 选择本目录：

```text
D:\Desktop\codex\Project\商品列表_files\category-correction-extension
```

6. 安装后，浏览器右上角会出现「店小秘插件」。

修改代码后，需要在 `chrome://extensions/` 页面点击「重新加载」才能生效。

## 使用说明

### 店小秘按数量选择

1. 打开店小秘数据搬家页面：

```text
https://www.dianxiaomi.com/web/productCrawl/productBanjia/popTemu
```

2. 确认列表已经加载出商品。
3. 点击浏览器右上角「店小秘插件」。
4. 在「店小秘选择数量」里输入数量，例如 `100`。
5. 点击「按数量选择」。
6. 等待日志显示选择完成。

说明：

- 选择单位是商品行，不是 SKU 行。
- 如果一页不足目标数量，插件会自动翻到下一页。
- 如果没有下一页或页面加载超时，会停止并在日志里显示实际选择数量。

### 分类建议采集

1. 打开带有「商品分类错误待修正」标签的商品列表页。
2. 点击浏览器右上角「店小秘插件」。
3. 点击「采集」。
4. 等待插件读取页面和 hover 弹窗。
5. 在文本框中查看 Markdown。
6. 点击「复制 Markdown」复制结果。

输出示例：

```md
## 商品分类修正建议

全部 SKC：36297562766,66523859733

### 1. 商品标题
- SKC：36297562766
- 原先错误类目：运动与户外用品 > 户外休闲 > 露营、徒步用品 > 户外炊具
- 修改建议：
  1. 家居、厨房用品 > 厨房和餐厅 > 厨房用具和小工具 > 烹饪用具 > 钳
  2. 家居、厨房用品 > 厨房和餐厅 > 厨房用具和小工具 > 烧烤工具 > 烧烤钳
  3. 家居、厨房用品 > 厨房和餐厅 > 厨房用具和小工具 > 烧烤工具套件
```

## 注意事项

- 插件只处理当前浏览器页面已经渲染出来的内容。
- 分类建议依赖页面 hover 弹窗，如果网页没有把「当前类目 / 修改建议」渲染到 DOM，插件无法凭空读取。
- 店小秘按数量选择依赖 `vxe-table` 表格结构，如果店小秘页面结构大改，需要同步调整选择逻辑。
- 使用前请先确认筛选条件、页码和列表内容正确，避免批量勾选到不想处理的商品。
